# MyHttpServer 

A simple HTTP server project built for learning and experimentation.  
This project demonstrates how a basic web server works, serving HTML content to a browser.

---

## How It Works

This project creates a simple HTTP server that:

1. Listens on port 8080
2. Waits for incoming requests
3. Responds with an HTML (home.html in this repo)
